## 审计脚本局限性

当前审计未能检测到的问题：
1. **图片内容质量**: 只能检查文件是否存在，不能检查内容是否完整
2. **选项语义正确性**: 只能检查是否为空/占位符，不能检查是否与PDF一致
3. **表格数据准确性**: 只能检查结构，不能检查数值是否正确
4. **答案正确性**: 只能检查是否存在，不能检查是否正确
